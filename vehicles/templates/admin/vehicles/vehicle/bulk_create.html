Instructions

  • Operator: Select the operator that will own these vehicles
  • Source: Select the data source for tracking where these vehicles came from
  • Vehicle Codes: Enter one vehicle code per line. These will be used as the vehicle's code field and combined with the operator to create unique slugs
  • Fleet Number Start: If provided, fleet numbers will be assigned starting from this number and incrementing for each vehicle
  • Vehicle Type: If selected, all created vehicles will be assigned this vehicle type
  • Livery: If selected, all created vehicles will be assigned this livery

Example Vehicle Codes:

YY67_HBG
YJ18_DHC
YJ18_DHD
SJ73_HSV
SJ73_HSX

Note: Vehicles with duplicate slugs (operator + code combination) will be skipped to avoid conflicts.
